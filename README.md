1. Clonar el proyecto
2. Instalar las dependencias `npm install`
3. Cambiar el archivo ``.env.template`` a ``.env``
4. Llenar las variables de ambiente
5. Levantar la DB
```bash
docker-compose up -d
```
7. Ejecutar el proyecto ```npm run start:dev```
